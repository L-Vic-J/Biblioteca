# Estructuras de Datos Aplicadas

Proyecto académico orientado a la aplicación práctica de estructuras de datos lineales, utilizando escenarios reales en contextos educativos, tecnológicos y comunitarios.  
El repositorio presenta distintos casos de estudio que permiten analizar y justificar el uso de cada estructura según el problema planteado.

## Objetivo
Aplicar los conceptos fundamentales de estructuras de datos mediante el desarrollo de soluciones prácticas, fortaleciendo la lógica, el análisis y la correcta selección de estructuras como listas enlazadas, pilas y colas.

## Escenarios desarrollados

## Sistema de Gestión de Biblioteca
- Lista enlazada simple: catálogo de libros.
- Lista doble circular: usuarios (estudiantes y docentes).
- Pila: historial de préstamos por usuario.
- Cola: lista de espera para libros no disponibles.



## Plataforma de Aprendizaje Virtual
- Lista simple circular: navegación continua entre cursos.
- Lista enlazada simple: módulos de cada curso.
- Pila: actividades pendientes de los estudiantes.
- Cola: solicitudes de soporte técnico.

## Sistema de Gestión de Proyectos Colaborativos
- Lista doble circular: integrantes del proyecto.
- Cola: asignación de tareas por orden de llegada.
- Pila: historial de modificaciones del proyecto.
- Lista enlazada simple: hitos y entregables.

---

## Aplicación de Reciclaje Comunitario
- Lista doble circular: participantes del programa.
- Lista simple circular: rutas de camiones recolectores.
- Cola: solicitudes de recolección.
- Pila: historial de entregas por participante.
- Lista enlazada simple: catálogo de materiales reciclables.

---

## Estructuras de datos utilizadas
- Listas enlazadas simples  
- Listas simples circulares  
- Listas dobles circulares  
- Pilas (Stack)  
- Colas (Queue)  

---

## Cómo ejecutar
1. Abrir el proyecto en un IDE de Java (IntelliJ IDEA, NetBeans o Eclipse).
2. Ejecutar la clase principal correspondiente a cada escenario.
3. Seguir las instrucciones mostradas en consola.

---

## Notas
- No se utilizan estructuras de datos precompiladas de Java (`List`, `ArrayList`, `Vector`, arreglos, etc.).
- Todas las estructuras se implementan manualmente siguiendo el modelo Estructura – Nodo – Dato.
